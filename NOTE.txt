Source: git://git.openwrt.org/12.09/openwrt.git

CentOS 6 build prerequisite: 
	gettext flex bison gcc gcc-c++ zlib-devel ncurses-devel quilt \
	perl-Compress-Zlib perl-ExtUtils-MakeMaker

Build steps:
* merge feeds
* create symlinks (dl,patches)
* ./scripts/feeds update -a
* quilt push -a
* ./scripts/feeds update -i
* ./scripts/feeds install -a
* copy and edit config
* make defconfig or make menuconfig
* make
